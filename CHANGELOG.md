# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.0] - Initial release
- Core features: Zombie and Shadow API detection
- Route normalization for numeric IDs and UUIDs
- Git blame integration for accountability (author, date, file:line)
- CLI `api-gravekeeper scan` with `--code-dir` and `--log-file` flags
- Basic support for Flask `@app.route()` patterns
